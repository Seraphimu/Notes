### 什么是Git？

微软翻译
Git 是一个版本控制系统。

Git 可帮助您跟踪代码更改。

Git 用于协作处理代码。

- Git是版本控制系统。
- Git可以帮你标识已改动的代码。
- Git可以用来协作开发。
